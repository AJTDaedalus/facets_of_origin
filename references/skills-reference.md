# Available Skills Reference

Skills relevant to Facets of Origin's three pillars: ruleset design, PHB writing, and software tooling.

## Game Design & Mechanics

| Skill | Source | When to Use |
|---|---|---|
| `/game-design` | Standalone | Core TTRPG design theory — MDA framework (Mechanics, Dynamics, Aesthetics), core loops, difficulty curves, player experience optimization |
| `/systems-thinking-leverage` | Lyndonkl | Analyze how game mechanics interact, find unintended consequences, identify where one rule change has maximum impact across the system |
| `/constraint-based-creativity` | Lyndonkl | When 2d6 resolution design, class balance, or rule mechanics feel stuck — turn the constraints of a lightweight system into creative fuel |
| `/prototyping-pretotyping` | Lyndonkl | Test mechanic ideas cheaply before full implementation — paper prototypes, concierge playtests, Wizard-of-Oz testing |
| `/design-evaluation-audit` | Lyndonkl | Systematic evaluation of mechanics clarity, balance, and presentation before PHB publication |
| `/evaluation-rubrics` | Lyndonkl | Create consistent scoring criteria for rule clarity, mechanic balance, or playtest feedback analysis |
| `/heuristics-and-checklists` | Lyndonkl | Build pre-playtest checklists, rapid balance assessment frameworks, and error-prevention heuristics for design decisions |
| `/encounter-balance` | Standalone | Evaluate and balance combat encounters — difficulty scaling, action economy, CR-equivalent analysis for the 2d6 system |
| `/fantasy-world-building` | Standalone | Fantasy genre conventions, magic system design patterns — useful for Shattered Origin setting design and lineage creation |

## PHB Writing & Organization

| Skill | Source | When to Use |
|---|---|---|
| `/information-architecture` | Lyndonkl | Organize PHB content for discoverability — chapter taxonomy, navigation structure, findability optimization so players can find rules quickly |
| `/cognitive-design` | Lyndonkl | Design visual layouts, information presentation, and educational content for the PHB and digital toolset UI |
| `/writing-structure-planner` | Lyndonkl | Plan overall PHB chapter structure — select structure types, map material flow, place key moments |
| `/writing-revision` | Lyndonkl | Three-pass editing system (cut clutter, reduce cognitive load, improve rhythm) for PHB prose |
| `/writing-pre-publish-checklist` | Lyndonkl | Final quality gate before publishing — content accuracy, structure, clarity, style consistency, polish |
| `/continuity-check` | Standalone | Cross-reference PHB chapters for contradictions — rule inconsistencies, terminology drift, orphan references |
